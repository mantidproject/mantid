=======
Guinier
=======

.. index:: Guinier

Description
-----------

The Guinier model is defined by:

.. math:: I(q) = I_0 \exp{(-R_g^2 q^2 / 3)}

where:
-  :math:`I_0` is the amplitude
-  :math:`R_g` is the radius of gyration
