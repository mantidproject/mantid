.. _func-Lorentz:

=======
Lorentz
=======

.. index:: Lorentz

Description
-----------

The Ornstein-Zernicke model is defined as:

.. math:: I(q) = I_0 / (1 + q^2L^2) + background

where:
-  :math:`I_0` is a scale factor
-  :math:`L` is the screening length
-  :math:`background` is the background level
