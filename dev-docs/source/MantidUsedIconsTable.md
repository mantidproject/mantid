# Mantid Icon Table

Here is a list of all currently used icons in Mantid. These icons are
from Material Design Icons (3.6.95), an open source icon project by
Google. Best place to find new unused icons is on the
[cheatsheet.](https://cdn.materialdesignicons.com/3.6.95/)

A developer who adds a new icon to a GUI should update this list with
it's specific usage if it can be used anywhere else in the codebase.
