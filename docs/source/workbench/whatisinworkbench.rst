===========================
What is there in Workbench?
===========================

Most of what users currently use in MantidPlot is or will be incorporated into
workbench. This includes but is not limited to, user group interfaces,
scripting window (fully integrated into the main window), workspace display,
algorithm widgets, and plotting.
