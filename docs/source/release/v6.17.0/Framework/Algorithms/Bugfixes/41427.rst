- :ref:`FileFinder <mantid.api.FileFinderImpl>` now recognises compound file extensions (e.g. ``.nxs.h5``) listed in ``Facilities.xml`` when expanding multi-run hints, so ranges such as ``CNCS100:105.nxs.h5`` resolve correctly instead of being misinterpreted as a stem ending in ``.nxs`` plus an ``.h5`` extension. As a consequence, a hint that explicitly specifies one extension (e.g. ``BSS_24234_event.nxs`` or ``SXD30904.raw.md5``) no longer silently resolves to a file with a different extension; specify the run number without an extension to let the facility's preferred extensions apply.
