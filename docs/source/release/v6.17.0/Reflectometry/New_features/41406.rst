- A new plotting tab has been added to the :ref:`ISIS Reflectometry Interface <interface-isis-refl>`. It lists successful reduction outputs by group and run, identifies each item and output type, supports plotting selected workspaces individually, overlaid, or tiled, and shows output workspace groups with their member workspaces as selectable child entries. It also includes reflectivity, detector map, spin asymmetry, and alignment plot output presets.
