DNS Powder TOF
==============

.. image::  ../../../images/DNS_interface_powder_tof_overview.png
   :align: center
   :height: 400px

\

data for powders. It can be accessed from the main menu of the "DNS
Reduction" GUI by clicking on "Tools" â†’ "Change Mode" and selecting
the "Powder TOF" option.

Below the interface menu, there are various tabs for data processing.
These tabs should be called from left to right during the data processing.
