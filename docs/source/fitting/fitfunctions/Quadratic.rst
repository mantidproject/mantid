=========
Quadratic
=========

.. index:: Quadratic

Description
-----------

A quadratic background function is defined as:

.. math:: \mbox{A0}+\mbox{A1}*x+\mbox{A2}*x^2

where

-  A0 - coefficient for constant term
-  A1 - coefficient for linear term
-  A2 - coefficient for quadratic term
