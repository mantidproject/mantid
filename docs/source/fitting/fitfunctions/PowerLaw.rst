.. _func-PowerLaw:

================
PowerLaw
================

.. index:: PowerLaw

Description
-----------

A power law may be used to describe:

.. math:: y = ax^b + c

where:
- a - magnitude
- b - exponent
- c - is a constant

Examples
--------

An example of when this might be used is for spin relaxation modelling [1].
