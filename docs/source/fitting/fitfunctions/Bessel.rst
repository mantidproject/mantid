==============================
Bessel
==============================

.. index:: Bessel

Description
-----------

Fitting function for use by Muon scientists defined by:

.. math:: 

	A(t)&=A_0J_0(2\pi\nu t+\phi)




.. attributes::

.. properties::
$J_0(x)$ is , $A_0$ is the amplitude at $t=0$, $\nu$ is wavenumber , $\phi$ is the phase.
