Description
-----------

This algorithm performs integration of single-crystal peaks within a
radius (with optional background subtraction) in reciprocal space.

The V1 of the algorithm is deprecated and left for compartibility with the scripts, which have the property
*CoordinatesToUse* set. Use :ref:`algm-IntegratePeaksMD-v2` for any new scripts.
